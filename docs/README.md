# libEdu Documentation

This folder centralizes operational and implementation documentation that was previously at the repository root.

## Structure

- `admin/`: Admin panel guides and workflows.
- `database/`: Database setup and migration guidance.
- `performance/`: Performance tuning notes.
- `localization/`: Site localization and i18n references.
- `reference/`: Quick commands and short references.
- `implementation/`: Implementation logs and summaries.
- `comparison/`: Before/after comparison documents.
- `environment/`: Environment and local domain setup.
- `examples/`: Example implementation artifacts.

## Conventions

- New project docs should be added under one of these categories.
- Keep root-level files focused on runtime/configuration only.
- If a new category appears repeatedly, add a dedicated subfolder.
